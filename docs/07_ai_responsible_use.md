# Sprint 1 Review

## 1. Sprint Review Purpose

The purpose of the Sprint 1 Review is to inspect the first increment of **AI Risk Detector** and collect feedback to improve the Product Backlog and future sprint work.

Sprint 1 focused on validating the core MVP concept:

> A user enters a software task or requirement and receives a structured AI-supported risk analysis.

The review evaluates whether this first increment demonstrates the main product value and whether the project is aligned with the assignment objectives.

---

## 2. Increment Presented

During Sprint 1, the team presented the initial MVP concept of AI Risk Detector.

The increment included:

- a defined product idea;
- a clear problem statement;
- an initial Product Backlog;
- selected MVP user stories;
- a first structured risk analysis example;
- initial Scrum documentation;
- initial repository organisation;
- Jira board preparation or setup.

The increment demonstrates how the assistant can support Agile and DevOps teams by identifying risks before implementation or release.
